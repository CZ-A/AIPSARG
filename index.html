Welcome to AIPSARG

AIPSARG adalah sebuah proyek berbasis Python untuk analisis dan otomatisasi trading menggunakan strategi PSAR.

Fitur Utama

  • Strategi PSAR dengan parameter yang dapat disesuaikan.
  • Mengambil data candlestick dan volume pasar.
  • Integrasi dengan OKX API untuk trading otomatis.

Instalasi

git clone https://github.com/CZ-A/AIPSARG.git
