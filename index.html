About AIPSARG

AIPSARG is an automated trading bot designed to execute PSAR-based strategies efficiently on OKX exchange. It is built for traders seeking reliability, precision, and speed.

Features

  • Customizable PSAR parameters for adaptive trading strategies.
  • Real-time market data monitoring and analysis.
  • Comprehensive logging and reporting of trade performance.
  • Secure integration with OKX API.

Setup

  1. Clone the repository: git clone https://github.com/cz-a/git
  2. Install dependencies: pip install -r requirements.txt
  3. Configure your .env file with API keys and parameters.
  4. Run the bot: python main.py
